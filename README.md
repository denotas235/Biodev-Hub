## 🎯 Visão Geral

### O que é?

BioDev Hub é um **aplicativo desktop/mobile** que ajuda programadores a:

- ✅ Resolver problemas **sem perder o que construíram**
- ✅ Gerenciar **ideias criativas** (código, arte, música, shaders)
- ✅ Integrar **TODAS as IAs disponíveis** em um só lugar
- ✅ Conectar com **GitHub** de forma inteligente
- ✅ Organizar tudo **automaticamente** com sistema biorelativológico

### Para quem é?

- Programadores profissionais
- Desenvolvedores de mods (Minecraft, jogos)
- Criadores de shaders e gráficos
- Quem programa + cria + organiza
- Quem quer **evoluir sem quebrar**

## 🔬 Método Biorelativológico

### O núcleo do sistema:

Input → Processing → Analyzing → Output (ciclo contínuo)

### Etapas:

1. **Observação/Fé** — ver/sentir tudo de todos ângulos
2. **Hipoterelação** — relacionar hipóteses contraditórias
   - Hipoteafirmação (especulação afirmativa)
   - Hiponegação (especulação negativa)
3. **1ª Analítica** — predição analítica (não é definitiva!)
4. **1ª Conclusão** — morte prematura (cuidado!)
5. **2ª Analítica** — exame de execução (verificação rigorosa)
6. **2ª Conclusão** — certidão de óbito (quase-final)
7. **Ponto Nulo** — equilíbrio (-1+1=0, caos+paz=equilíbrio)
8. **Cadeia de Denorium** — problemas em cadeia (A gera B gera C)

### Relasofia / Anthoporelasofia:

- **Conhecimento** — base para o conhecimento
- **Dúvida Cognitiva** — quem duvida sabe que não sabe
- **Reflexão Crítica** — só refletimos o que conhecemos
- **Razão** — verdade relativa (não absoluta!)

## 🛠 Como Usar

1. **Clonar o repositório:**
   ```bash
   git clone https://github.com/denotas235/Biodev-Hub.git
   cd Biodev-Hub
   ```

2. **Instalar dependências (Node):**
   ```bash
   npm install
   ```

3. **Executar o app:**
   ```bash
   npm start
   ```

4. **Contribuir:**
   - Fork o projeto
   - Cria branch: `explore-hipoteafirmação-TUA-IDÉIA`
   - Commit changes
   - Push para branch
   - Cria Pull Request
